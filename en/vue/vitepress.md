# vitepress

install

```sh
npm add -D vitepress
npx vitepress init
```

## The Config File

The config file `.vitepress/config.js`

```js
// .vitepress/config.js
export default {
  // site-level options
  title: 'VitePress',
  description: 'Just playing around.',

  themeConfig: {
    // theme-level options
  }
}
```
