# vim command

* to the line end `$` = `ctrl + 4`
* to the line end `^` = `ctrl + 6`
* delete the file content `ggdG`
* copy current line `yyp`
* save and exit `:wq`
* to line `:number` such `:12`
* `h`(left) `j`(up) `k`(down) `l`(right)
* `w` next word `b` previous word
* `f` to the word
* `x` delete a char
* `dd` delete current line
* `v` select word
* `u` undo
* `cw` change word
* `:vs` split right
* `:sp` split down
* `ctrl + w +j` move cursor to down
* `ctrl + w +h` move cursor to left
* `ctrl + w +l` move cursor to right
* `ctrl + w +k` move cursor to up
* `ctrl + w +w` move cursor to next window
* `ctrl + F` new page
* `qa`  record
* `: normal @a` do the all record
* `:e filename` open file
* `ZZ` save and quit
